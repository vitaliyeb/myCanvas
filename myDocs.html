canvas.getContext(contextType, contextAttributes) - возвращает контекст рисования на холсте, или null, если идентификатор контекста не определен.

  • contextType - строка содержащая идентификатор контекста рисования связанного с холстом
  • "2d" - создает объект CanvasRenderingContext2D, представляющий двумерный контекст
  • "webgl" - создает объект WebGLRenderingContext, представляющтй трёхмерный контекст. Этот контекст доступен только в браузерах, которые реализуют WebGL первой версии (OpenGL ES 2.0).
  • "webgl2" - создает объект WebGL2RenderingContext, представляющтй трёхмерный контекст. Этот контекст доступен только в браузерах, которые реализуют WebGL второй версии (OpenGL ES 3.0).
  • "bitmaprenderer" - создает ImageBitmapRenderingContext, обеспечивающий только функции для замены содержимого холста с заданным ImageBitmap.
  • contextAttributes - объект с настроками контекста
  • 2d атрибуты контекста:
  • alpha - логическое значение, указывает есть ли у холста альфа канал
  • Атрибуты контекста WebGL:
  • alpha - Boolean, который указывает, содержит ли холст альфа-буфер
  • depth - Boolean, который указывает, что буфер рисования имеет буфер глубины не менее 16 бит.
  • stencil - Boolean, который указывает, что буфер рисования имеет буфер трафарета не менее 8 бит.
  • antialias - Логическое значение, указывающее, выполнять ли сглаживание или нет.
  • premultipliedAlpha - Boolean, который указывает, что компоновщик страниц будет предполагать, что буфер рисования содержит цвета с предварительно умноженной альфа.
  • preserveDrawingBuffer - Если значение равно true, буферы не будут очищены и сохранят свои значения до тех пор, пока автор не очистит или не перезаписает их.
  • failIfMajorPerformanceCaveat - Boolean, который указывает, будет ли создан контекст, если производительность системы низкая.
  • Метод возвращает:
  • RenderingContext, который представляет собой
  • CanvasRenderingContext2D для "2d"
  • WebGLRenderingContext для "webgl" и "experimental-webgl"
  • WebGL2RenderingContext для "webgl2" и "experimental-webgl2"
  • ImageBitmapRenderingContext для "bitmaprenderer"

canvas.fillRect(posX, posY, width, height) - Рисует закрашенный прямоугольник.

  • posX - Позиция начала рисования по оси X
  • posY - Позиция начала рисования по оси Y
  • width - Ширина прямоугольника
  • height - Высота прямоугольника

canvas.strokeRect(posX, posY, width, height) - Рисует контур прямоугольника.

  • posX - Позиция начала рисования по оси X
  • posY - Позиция начала рисования по оси Y
  • width - Ширина прямоугольника
  • height - Высота прямоугольника

canvas.clearRect(posX, posY, width, height) - Очистка прямоугольной области, делая содержимое совершенно прозрачным.

  • posX - Позиция начала рисования по оси X
  • posY - Позиция начала рисования по оси Y
  • width - Ширина прямоугольника
  • height - Высота прямоугольника

canvas.beginPath() - Начало рисования новой фигуры;

canvas.moveTo(posX, posY) - Перемещает перо в точки X и Y;

  • posX - Позиция перемещения пера по оси X
  • posY - Позиция перемещения пера по оси Y

canvas.fill() - Отрисовывает закрашенную фигуру;

canvas.stroke() - Отрисовывает фигуру по контуру;

canvas.closePath() - Закрывает путь от начальной точки до конечной давая возможность инициализировать новый.;

canvas.moveTo(posX, posY, Radius, StartAngle, endAngle, anticlockwise) - Перемещает перо в точки X и Y;

  • posX - Позиция перемещения пера по оси X
  • posX - Позиция перемещения пера по оси X
  • Radius - Радиус окружности
  • StartAngle - Угол начала дуги
  • endAngle - Угол конца дуги
  • anticlockwise - Необязательный Boolean. Задаёт направление рисования дуги.
  • true - по часовой
  • false - против часовой
